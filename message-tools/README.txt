Message Utilities

From the console, run:
java -jar message-tools.jar

Usage: message-sender [--help] [-c=<count>] [-h=<host>] [-p=<port>] <command>
Tools for the REST API exercise.
      <command>         generate-messages or generate-requests
      --help            display this help and exit
  -c, --count=<count>   number of messages to generate
  -h=<host>             the destination host
  -p=<port>             the destination port
